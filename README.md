# Crazy Driver

A 3D endless runner built in Godot 4. Weave through city traffic across 3 lanes,
collect coins, jump over obstacles, and unlock new car colors — all while the
world accelerates around you.

## How to play

| Key | Action |
|-----|--------|
| ← → | Switch lanes |
| Space | Jump |
| Enter | Start / restart |
| P | Open car shop |
| Esc | End run |

## Features

- Procedurally generated road, buildings, and traffic — no imported 3D assets
- Speed increases over time, shortening jump windows and tightening spawn intervals
- 8 unlockable car colors purchasable with coins earned across all runs
- Coins carry over between games and never reset
- Shop can be opened mid-run (game pauses automatically)
